Πρώτο βήμα

Είστε ένας ενήλικας που έχει αποφασίσει να κάνει μαθήματα Αγγλικών. Ποιό είναι το πρώτο βήμα;

Αρχικά, πατήστε το κουμπί παρακάτω για να επικοινωνήσετε μαζί μας.

Θα σας οδηγήσει σε μια ηλεκτρονική φόρμα που θα συμπληρώσετε για να μας δώσετε πληροφορίες σχετικές με το ενδιαφέρον σας, και τα στοιχεία επικοινωνίας σας.

Επίσης, θα μας ενημερώσετε με ποιό τρόπο επιθυμείτε να επικοινωνήσουμε μαζί σας.

Φόρμα Επικοινωνίας

Σε Εξέλιξη

Μόλις υποβάλετε την φόρμα, θα επικοινωνήσω μαζί σας το συντομότερο δυνατόν.

Σε 2-5 εργάσιμες μέρες, θα επικοινωνήσω μαζί σας για να συζητήσουμε λεπτομερώς τις ανάγκες σας και το πως μπορώ να σας βοηθήσω. Θα μιλήσουμε για τα σχέδια σας, τις ανάγκες σας, το πρόγραμμα σας, την αμοιβή και άλλες σχετικές λεπτομέρειες.

Εάν συμφωνήσουμε ότι οι υπηρεσίες που μπορώ να προσφέρω καλύπτουν τις προσωπικές σας ανάγκες, ή τις ανάγκες του γκρουπ σας, Θα προχωρήσουμε στο επόμενο βήμα.

Προετοιμασία για μαθήματα από απόσταση

Θα συζητήσουμε σχετικά με τα εργαλεία που θα χρησιμοποιήσουμε για τα μαθήματα.

Θα συζητήσουμε σχετικά με τους λογαριασμούς που θα χρησιμοποιήσουμε για την επικοινωνία μας, για την διδασκαλία, και για την ανταλλαγή πληροφοριών και διδακτικού υλικού.

Είμαι ευέλικτος σχετικά με τα διαδικτυακά εργαλεία, οπότε μπορούμε να χρησιμοποιήσουμε εφαρμογές που γνωρίζετε και νιώθετε άνετα να χρησιμοποιείτε. Επίσης, θα υπάρχει υποστήριξη αν αντιμετωπίζετε τεχνικά προβλήματα ή αν δεν είστε εξοικειωμένοι με την τεχνολογία.

Προετοιμασία για το πρώτο μάθημα

Αφού έχουμε συμφωνήσει για τα παραπάνω, προετοιμαζόμαστε για το πρώτο μάθημα.

Σε αυτό το στάδιο, πρέπει να είμαστε έτοιμοι να ξεκινήσουμε:

Θα έχουμε συμφωνήσει όλοι μεταξύ μας το πρόγραμμα και τις ώρες μαθήματος

Αν υπάρχει ανάγκη για αγορά διδακτικού υλικού, καλό θα είναι να έχει ολοκληρωθεί (βιβλία, τετράδια, κτλ)

Θα έχουμε κάνει δοκιμή για να βεβαιωθούμε ότι λειτουργούν σωστά οι εφαρμογές τηλεδιάσκεψης στις συσκευές σας

Θα έχουμε συζητήσει όλες τις λεπτομέρειες που χρειάζονται για να έχουμε ομαλά μαθήματα (πχ, σύνδεση στο ίντερνετ, προβλήματα με εφαρμογές, κτλ)

Θα υπάρχει υποστήριξη για ότι προβλήματα αντιμετωπίζετε

Αν δεν είμαστε έτοιμοι για το πρώτο μάθημα, θα πάρει περισσότερο χρόνο για να ξεκινήσετε να μαθαίνετε:
Γι'αυτό, μπορούμε να συνεργαστούμε πριν ξεκινήσουμε για να βεβαιωθούμε ότι τίποτα δεν θα διακόψει τα μαθήματα μας.
